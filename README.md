# my-react-boilerplate

My own React Boilerplate

- webpack-dev-server, browser-sync, hot module replacement
- react, redux, redux-thunk
- react-router-4
- immutability-helper
- eslint
- less, css-modules, extract-text-webpack-plugin
- gulp for prod tasks

--------------

## Installation

`yarn install`

--------------

## Start dev server

`npm start`

--------------

## Run build task

`npm run build`
